# Agentic Operations UI Reskin

Date: 2026-05-18.

## Goal

Reframe the dashboard from a SOC-specific console into the broader autonomous
enterprise operations control plane without changing navigation, API contracts,
or dashboard behavior.

The product story is now visible as **Agentic Operations** / **AGENTIC OPS**.
SOC and security remain supported capability areas inside the platform, not the
only product boundary.

## Scope

This was intentionally a light reskin:

- Changed the browser title from `SOC Dashboard` to `Agentic Operations`.
- Replaced the sidebar radar/crosshair mark with a small operations graph mark.
- Changed the sidebar product label from `SOC COMMAND` to `AGENTIC OPS`.
- Renamed visible sections:
  - `Service Desk Intake` -> `Request Intake`
  - `CI/CD Security Gates` -> `CI/CD Delivery Gates`
  - `Recent Security Runs` -> `Recent Delivery Runs`
  - `SOC Tools` -> `Provider Modules`
  - `AI Endpoint` -> `Model Gateway`
- Broadened intake placeholder copy to include access, outages, deployments,
  alerts, and suspicious email as examples.
- Kept the existing tab set: Overview, Tickets, Intake, Agents, Changes,
  Workflows, Postmortems, CI/CD, Learning, Tools, Setup, Access, Audit.

## Files Changed

- `frontend/index.html`
- `frontend/js/dashboard.js`
- `frontend/css/dashboard.css`

## Intentional Remaining Strings

The following classes of strings may still include `SOC` or security wording:

- Compatibility globals such as `SOC_DASHBOARD_CONFIG`.
- Comments in JS module headers.
- Legitimate domain/provider labels, such as `IT/SOC Operations Baseline`.
- Security-specific tickets, workflows, or module data rendered from the API.

Do not rename backend/API identifiers or data contracts as part of this reskin.

## Verification

Run:

```powershell
rg -n "SOC|Security Runs|Security Gate|phishing|SOC Dashboard|SOC COMMAND" frontend
```

Expected remaining hits should only be compatibility names, comments, or
legitimate security/SOC module references. Visible first-screen framing should
read as an agentic operations platform.
